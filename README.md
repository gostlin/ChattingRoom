### 简易聊天室

---

1. 进去 socket 安装依赖 npm install/yarn install

2. yarn run dev 启动服务端

3. 进入 client 安装依赖 npm install/yarn install

4. yarn run dev 启动客户端

5. 通过 url 的 username 设置用户的名字
